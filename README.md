# DADAO-SimRISC project
This is the main developing branch of DADAO architecture and SimRISC instruction system. It contains a collection of configuration tools and patches for constructing ```toolchains``` (gcc, llvm, binutils-gdb, newlib-cygwin, glibc) ```executing environment``` (qemu) ```benchmarking toolkits``` (dejagnu, testsuites) ```SoC implementation``` (chipyard) and ```OS``` (linux) for DADAO & SimRISC.
- Information about DADAO & SimRISC: check [DADAO.wiki](https://github.com/gxt/DADAO/wiki) documentation for more details.
